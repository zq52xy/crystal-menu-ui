# Design Prompt: Original JRPG Blue Menu System

Design an original classic JRPG blue menu UI system for a learning-only component library.

The design should include:

- command menu,
- title/main menu,
- party/menu shell layout,
- confirmation popup,
- party status panel,
- dialogue box,
- inventory list,
- item tooltip,
- save slot list,
- equipment slots and stat comparison panel,
- standalone orb gems,
- large linked orb socket rail,
- battle command shell,
- bottom floating status/action bar.

Visual language:

- dark glass gradient panels,
- thin cyan borders,
- white text,
- cyan selection highlight,
- angular level/status geometry,
- pulsing selected-row highlight,
- compact playtime/currency/action metadata,
- compact equipment rows, stat deltas, and linked orb sockets,
- standalone green, violet, gold, red, cyan, and empty CSS-rendered orb gems,
- large horizontal blue socket rails that reuse the same gem cores,
- three-region command/status/detail layout with a bottom HUD strip,
- centered blue glass dialog with a diamond alert symbol and two horizontal actions,
- save/load archive rows with a left diamond slot marker, timestamp, location, playtime, party initials, and selected/autosave/empty states,
- original cyan icon-vector set for combat, magic, defense, stat, equipment, time, currency, and summon categories,
- character profile panels with large portrait slots, compact resource meters, stat chips, and original placeholder art when no licensed portrait is supplied,
- full profile screens with roster, focused profile, details, commands, footer metadata, and replaceable portrait slots,
- compact cyan separators and loading indicators for system states,
- inventory lists with item icons, quantities, selected rows, detail tooltips, and battle command shells with ATB/Limit gauges,
- compact grid-based spacing,
- restrained red/green state accents,
- original symbols only.

Do not include official game logos, characters, screenshots, fonts, icons, music references, remote Figma assets, or direct screen recreations.
