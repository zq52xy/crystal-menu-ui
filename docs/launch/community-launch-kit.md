## Reddit / Forum Version

I made an MIT-licensed React component library for original JRPG-style glass HUDs. The goal is not to ship game assets, but to turn common RPG menu patterns into typed components and AI-readable docs.

It includes 30 components for equipment screens, party status, battle commands, orb/socket systems, limit gauges, damage numbers, and cinematic chapter cards. All visuals are CSS/SVG based and de-branded.
