# Vite Basic Example

Minimal Vite + React + TypeScript app for trying `crystal-menu-ui` from a fresh repository clone.

## Run

```bash
cd examples/vite-basic
npm install
npm run dev
```

## Check

```bash
npm run build
npm run preview
```

## Local Development Note

`vite.config.ts` maps `crystal-menu-ui` and `crystal-menu-ui/style` to the repository source files. This keeps the example runnable before the package has been built or published locally.
