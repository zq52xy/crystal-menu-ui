# Module Map: examples

## Purpose

Runnable consumer examples that show how to use `crystal-menu-ui` from real app shells.

## Members

- `vite-basic/`: minimal Vite + React + TypeScript app that imports package-like entry points through local source aliases for repository development.

## Boundaries

- Examples must use public package exports only.
- Examples may add app-specific layout styles, but shared component styling stays in `src/`.
- Examples must not include official screenshots, logos, character art, extracted assets, or remote proprietary references.
